Event Tutorial

To start using the eventsystem you need to follow a few simple steps.
first you need to create a new script and place it on any gameobject in the scene.
from this script, remove the update and start functions.

to start recieving events you first need a function that will be activated when a event happens.
this function will need to have 2 parameters, a GameObject and a float.

void SomeFunction(GameObject g, float f)
{
    //code for linked event goes here
}


now that we have a function that we want to be called by the event, we have to make the EventManager know about this.
for now we do this in the OnEnable of the script by calling EventManager.StartListening. this function also takes 2 parameters.
these parameters are the event name and the function to be called.

void OnEnable()
{
    EventManager.StartListening("eventnamehere", SomeFunction);
    //add all events that you want to listen to when the game starts here.
}


to make sure that we do not create memory leaks you have to do one more thing.
for everytime you call EventManager.StartListening() you have to also call EventManager.StopListening in the OnDisable.
this function again takes 2 parameters: the event name and the function that should stop being called.

void OnDisable()
{
    EventManager.StopListening("eventnamehere", SomeFunction);
    //add all events to stop listening to here
}


now you are ready to script to your first event!
